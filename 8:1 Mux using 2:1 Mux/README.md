# 8:1 MUX using 2:1 MUX

The 8:1 MUX contains 8 inputs, 3 select lines and 1 output. The 2:1 MUX contains 2 inputs 1 select line and 1 output. 
8:1 MUX is designed using seven 2:1 MUXS.
